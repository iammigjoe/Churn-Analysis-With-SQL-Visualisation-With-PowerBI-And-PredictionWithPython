Analyse du Churn — SQL • Power BI • Python (KNN)

Ce projet combine SQL, Power BI et Python pour analyser le churn client, visualiser les insights clés et prédire les départs à l’aide d’un modèle KNN.

Technologies Utilisées
• SQL (PostgreSQL) – requêtes avancées, nettoyage et préparation des données
• Power BI
• Power Query (M) – transformation des données, fusion, typage, nettoyage
• DAX – mesures, KPIs, calculs dynamiques
• Tableaux de bord interactifs
• Python (Pandas, NumPy, Scikit-learn) – preprocessing, classification KNN
• Windows VM sur Mac – configuration Power BI

Compétences Acquises
• ETL complet : extraction → nettoyage → transformation
• Manipulation avancée des données via Power Query
• Création de mesures et indicateurs avec DAX (ex : taux de churn, revenus moyens, segmentation)
• Développement de dashboards professionnels pour la prise de décision
• Préparation des données pour le machine learning
• Construction et évaluation d’un modèle KNN
• Résolution technique (exécuter Power BI sur macOS via machine virtuelle)
• Pipeline data end-to-end : SQL → BI → Machine Learning

Problème Métier Résolu
Le churn entraîne des pertes financières importantes pour les entreprises.
Ce projet permet de :
• Identifier les clients à risque de départ
• Comprendre les causes profondes du churn (prix, services, durée, support…)
• Anticiper les désabonnements avant qu’ils ne se produisent

Importance pour les Organisations
• Réduction directe de la perte de revenus
• Meilleures stratégies de fidélisation
• Décisions basées sur des indicateurs fiables (DAX + ML)
• Alignement entre équipes : marketing, finance, service client
• Passage d’une culture réactive à une culture prédictive

Skills Demonstrated
• Full ETL workflow using SQL + Power Query
• Advanced BI modeling with DAX
• Creation of KPI-driven dashboards for business decision-making
• Feature engineering and ML model building (KNN)
• Cross-platform problem solving (running Power BI on Mac)
• End-to-end data pipeline: SQL → BI → ML

Real-World Business Problem Solved
Churn causes major revenue losses in subscription-based industries.
This project addresses:
• Which customers are likely to leave?
• What factors drive churn?
